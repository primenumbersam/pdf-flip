PDF-to-Flip BookShelf

What is this?

This is a 3D Virtual Bookshelf showcasing static PDF documents converted into interactive flipbooks.
It uses pdf2image for processing and page-flip for the reading experience.

Python CSS 3D JS Flipbook

How to Check?

CLICK a book on the shelf below to open the viewer. Enjoy the tactile feel of turning pages!
